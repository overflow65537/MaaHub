# pipeline-option

本目录包含 MaaFramework runtime UI option 设计指南。

## 包含内容

- `SKILL.md`: option 使用规范与案例说明
- `references/protocol.md`: option 与 pipeline_override 协议参考
- `maahub_meta.json`: 网站元信息

## 说明

本 skill 说明如何在 MaaFramework 中新增 UI option，并确保 `assets/interface.json`、任务 option 注册、预定义 pipeline 节点三处联动正确。
